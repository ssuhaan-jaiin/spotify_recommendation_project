# 🎧 Spotify Recommendation Project

## 🚀 Overview
This project uses **unsupervised machine learning** to analyze Spotify track data and group similar songs together based on their audio features 🎵.

Techniques used include:
- Feature scaling  
- PCA (dimensionality reduction)  
- K-Means clustering  
- Data visualization  

The goal is to discover patterns in music and explore how clustering can help build recommendation systems.

---

## 🧠 Key Idea
Songs with similar audio characteristics are grouped into clusters, helping identify natural similarities in music.
